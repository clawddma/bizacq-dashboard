⚙ Auto-sync con GitHub

Cómo obtener el token (60 segundos):

  1. Abre github.com/settings/personal-access-tokens/new ↗
  2. Token name: bizacq-dashboard
  3. Expiration: 1 año (o lo que quieras)
  4. Repository access: "Only select repositories" → escoge bizacq-dashboard
  5. Permissions → Repository → Contents: Read and write
  6. Click "Generate token", copia el valor (empieza con github_pat_...)
⚠ Privacidad: el token queda guardado en localStorage de TU navegador. Nunca se commitea al repo. Solo lo ves tú en este equipo. Si comparte la URL del dashboard, los demás NO tienen tu token.
✓ Ya tienes un token guardado. Cambiar = pegar uno nuevo. Vaciar = deshabilitar auto-sync.
